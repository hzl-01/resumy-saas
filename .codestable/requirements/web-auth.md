# Web 服务与用户认证

## 用户故事

- 作为一个不懂命令行的求职者，我希望打开浏览器就能制作简历，而不是安装 Node.js 敲命令。
- 作为一个想快速修改简历的人，我希望登录后可以直接编辑已有的简历，而不是每次从头填。
- 作为团队里的非技术成员，我希望把 docx 简历拖到网页上就能变成排版精美的 PDF。

## 为什么需要

resumy 现有的 CLI 工具对于技术用户很好用，但对大部分人来说，"打开终端敲命令"这件事本身就劝退了。很多潜在用户的需求只是"选个模板 → 填内容 → 下载 PDF"——他们不需要 CLI 的能力，只需要一个网页。

## 怎么解决

通过 `resumy serve` 命令启动一个 Web 服务。用户在浏览器里注册账号、登录后，可以在网页表单里填写或上传简历，选择内置模板，一键生成并下载 PDF。所有数据保存在服务器上，下次登录还能接着改。

## 边界

- 不提供付费/订阅功能（MVP 免费）
- 不支持团队协作/共享简历
- 不支持 AI 辅助写简历内容
- 不提供简历投递/职位匹配
- 一期仅支持单语言
